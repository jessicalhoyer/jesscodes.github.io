Logbook Database

Summary

My Logbook Database is a personal project that stores information about my writing, including characters and writing snippets called "threads." This website allows me to search for characters and threads to read information about them. Each character and thread is stored in my local MySQL database and I use PHP and SQL to send queries to display results. In addition, I've made a management portal in the website which allows me to input characters and threads directly into the database instead of using the MySQL Workbench.


While this started off as a personal project, the skills that I learned with this greatly helped in creating my portion of the Buttweiler Environmental Project project. I was able to apply the skills I learned with database queries, pagination, and building a management portal into the project for the client.
